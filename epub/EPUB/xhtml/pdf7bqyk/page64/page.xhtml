Ejemplo 1

 

La suma de tres números consecutivos es 99. Halle los números.

 

Solución

 

Modelamos los números con literal

 

Primer número = x

 

Segundo número = x + 1

 

Tercer número = x + 2

 

De tal forma que:

 

x + x + 1 + x + 2  
=
99

 

3x + 3  
=
99

 

3x
=
99 − 3

 

 
3x =  
96

 

 
x =  
32

 

Por lo tanto, los tres números son 32, 33 y 34

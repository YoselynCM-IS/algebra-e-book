Con base a lo anterior, podemos decir que hay comporamientos que son

lineales o completamente proporcionales. Para este tipo de situaciones, se

puede manejar el algoritmo de la regla de tres simple.

damente problemas de proporcionalidad directa. Para hacer una regla de

3 simple necesitamos 3 datos: dos magnitudes proporcionales entre sí, y

una tercera magnitud. A partir de estos, averiguaremos el cuarto término

de la proporcionalidad.
